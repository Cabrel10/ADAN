PHASE 4: DÉPLOIEMENT SERVEUR (⏳ À FAIRE)
================================================================================

Étapes:

1. Créer l'archive de déploiement
   tar -czf adan_bot_deploy.tar.gz --exclude='.git' --exclude='__pycache__' --exclude='*.pyc' --exclude='logs/*' --exclude='.env' .

2. Transférer vers le serveur
   scp adan_bot_deploy.tar.gz user@serveur:/home/user/

3. Installer sur le serveur
   - Extraire l'archive
   - Créer l'environnement virtuel
   - Installer les dépendances
   - Configurer les variables d'environnement

4. Configurer le service systemd
   - Créer /etc/systemd/system/adan-bot.service
   - Activer le redémarrage automatique
   - Configurer les logs

5. Démarrer le service
   sudo systemctl start adan-bot
