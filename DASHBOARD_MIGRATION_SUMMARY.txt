TROUBLESHOOTING
================================================================================

Dashboard shows empty data?
  → Check if monitor is running: ps aux | grep paper_trading_monitor
  → Start monitor: python scripts/paper_trading_monitor.py

Dashboard shows old data?
  → Check state file: stat /mnt/new_data/t10_training/phase2_results/paper_trading_state.json
  → Restart monitor: pkill -f paper_trading_monitor

Connection error?
  → Create directory: mkdir -p /mnt/new_data/t10_training/phase2_results
  → Restart dashboard: python scripts/adan_btc_dashboard.py
